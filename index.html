Protein Function Prediction from UniProt Sequences

Qixuan Wu

Abstract

Protein function prediction has become an important area in bioinformatics, as experimental identification of protein roles is often slow and expensive. Computational methods can help bridge this gap by predicting protein function directly from sequences. In this project, a dataset of about 5,000 proteins was collected and cleaned from UniProt. The plan is to test different approaches, including machine learning models and newer deep learning models such as CNNs or Transformers, depending on the feasibility with the available resources. The results will be measured by accuracy and related evaluation metrics. The goal is to explore practical methods for sequence-based protein analysis and see which ones are most effective under the given dataset size and constraints.

Plan of Action

  • Dataset: UniProt/Swiss-Prot (link).
  • Data preprocessing: parse FASTA and XML, clean data, encode amino acid sequence.
  • Implementation:
    • Baseline classifier: Logistic Regression or simple feed-forward NN
    • Sequence models: CNN and/or a lightweight Transformer
  • Comparison: Compare model performance between baselines and deep learning models.
  • Experiments: Train/test split evaluation. Measure classification accuracy, precision, recall, F1-score.
  • Workload Distribution: All steps performed by Qixuan Wu
